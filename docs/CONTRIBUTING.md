# Contributing to the Global Payment Regulations Knowledge Base

Thank you for your interest in contributing! 

## How to Add a New Regulation
1. Go to the `templates/` folder and copy the `regulation.md` template.
2. Paste it into the correct `countries/[country-name]/` folder.
3. Rename the file to match the regulation (e.g., `payment-services-act.md`).
4. Fill out the YAML front matter at the top of the file.
5. Fill out all the mandatory sections in the body of the document.
6. Submit a Pull Request.

## Naming Conventions
- **Files:** Always lowercase, use hyphens for spaces (e.g., `anti-money-laundering.md`).
- **Folders:** Use official ISO country codes or lowercase names (e.g., `sg/` or `singapore/`).

## Need Help?
Open an Issue and select the "Enhancement" or "Bug Report" template.
